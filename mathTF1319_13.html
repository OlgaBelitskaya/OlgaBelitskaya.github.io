Task 1

1) Найдем область определения функции и точки разрыва.
$\large{\displaystyle{ y = \frac{4}{x^2 + 2 x - 3} = \frac{4}{(x - 1) \ (x + 3)} }}$
$\large{\displaystyle{ D(y)= (-\infty; -3) \cup (-3; 1) \cup (1; +\infty) }}$
$\large{\displaystyle{ x = -3; x = 1 }}$ - точки разрыва.
2) Вычислим односторонние пределы функции
- в граничных точках области определения,
- в точках разрыва.
Найдем вертикальные асимптоты.
$\large{\displaystyle{ \lim_{ x \to +\infty} \frac{4}{x^2 + 2 x - 3} = \lim_{ x \to +\infty} \frac{4}{x^2} = 0}}$
$\large{\displaystyle{ \lim_{ x \to -\infty} \frac{4}{x^2 + 2 x - 3} = \lim_{ x \to -\infty} \frac{4}{x^2} = 0}}$
$\large{\displaystyle{ \lim_{ x \to -3+0} \frac{4}{x^2 + 2 x - 3} = \frac{4}{(-3 + 0 - 1) \ (-3 + 0 + 3)} = \frac{4}{+ 0 \ (-4 + 0)} = -\infty }}$
$\large{\displaystyle{ \lim_{ x \to -3-0} \frac{4}{x^2 + 2 x - 3} = \frac{4}{(-3 - 0 - 1) \ (-3 - 0 + 3)} = \frac{4}{- 0 \ (-4 - 0)} = +\infty}}$
$\large{\displaystyle{ \lim_{ x \to 1+0} \frac{4}{x^2 + 2 x - 3} = \frac{4}{(1 + 0 - 1) \ (1+ 0 + 3)} = \frac{4}{+ 0 \ (4 + 0)} = +\infty}}$
$\large{\displaystyle{ \lim_{ x \to 1-0} \frac{4}{x^2 + 2 x - 3} = \frac{4}{(1 - 0 - 1) \ (1- 0 + 3)} = \frac{4}{- 0 \ (4 - 0)} = -\infty}}$
Прямые $x = -3, \ x = 1$ являются вертикальными асимптотами.
