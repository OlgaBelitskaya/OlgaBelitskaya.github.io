Решение / Decision¶

$\overline{P_1} = (x_1, x_2, x_3); \overline{P_2} = (x_4, x_5, x_6)$.

$\overline{P} = \overline{P_1} + \overline{P_2} \implies \begin{cases} x_1 + x_4 = 3 \\ x_2 + x_5 = 4 \\ x_3 + x _6 = 1 \end{cases} \iff \begin{cases} x_4 = 3 - x_1 \\ x_5 = 4 - x_2 \\ x_6 = 1 - x _3 \end{cases} \ (1)$

$\overline{P_1} \perp \overline{Q} \implies 1 * x_1 + 2 * x_2 + 1 * x_3 = 0 \implies x_1 = -2x_2 - x_3 \ (2)$

$\overline{P_2} \parallel \overline{Q} \implies \begin{cases} \lambda * x_4 = 1 \\ \lambda * x_5 = 2 \\ \lambda * x_6 = 1 \\ \end{cases} \ (3)$

(3) (1) (2):

$ \begin{cases} \lambda * (3 + 2x_2 + x_3) = 1 \\ \lambda * (4 - x_2) = 2 \\ \lambda * (1 - x_3) = 1 \end{cases} \iff \begin{cases} 3 + 2x_2 + x_3 = 1 / \lambda \\ 4 - x_2 = 2 / \lambda \\ 1 - x_3 = 1 / \lambda\\ \end{cases} $

:

$4 - x_2 - 3 - 2x_2 - x_3 - 1 + x_3 = 2 / \lambda - 1 / \lambda - 2 / \lambda \iff -3x_2 = 0 \iff x_2 = 0$

:

$3 + 2x_2 + x_3 - 1 + x_3 = 1 / \lambda - 1 / \lambda \iff 2x_3 = -2 - 2x_2 \iff x_3 = -1 -x_2 \implies x_3 = -1$

$x_1 = -2 * 0 -(-1) = 1$

$x_4 = 3 - 1 = 2; x_5 = 4 - 0 = 4; x_6 = 1 - (-1) = 2$

$\lambda = 1 / (1 - (-1)) = 0.5$

$\overline{P_1} = (1, 0, -1); \overline{P_2} = (2, 4, 2)$

$ABCD$ $AB$ $AD$, $C$.

$AD: 9X - 11Y + 2 = 0; AB: 11X - 9Y - 2 = 0; C = (21,21)$.

Решение / Decision¶

1) $AB$ $AD$ - $$:

$ \begin{cases} 9X - 11Y + 2 = 0 \\ 11X - 9Y - 2 = 0 \end{cases} $

11, - -9 :

$-40Y_A + 40 = 0 \implies Y_A = 1 \implies 9X_A - 11 + 2 =0 \implies X_A = 1 \implies A = (1, 1)$.

2) $AC$:

$\frac{X - X_A}{X_C - X_A} = \frac{Y - Y_A}{Y_C - Y_A}$

$\frac{X - 1}{21 - 1} = \frac{Y - 1}{21 - 1} \iff X -1 = Y - 1\iff X - Y = 0$

3) $AC$ $O$:

$X_O = (1 + 21) / 2 = 11; Y_O = (1 + 21) / 2 = 11 \implies O = (11,11)$

4) $BD \perp AC; O \in BD$. $BD$ ,

$k_{AC} * k_{BD} = -1$:

$k_{AC} = 1 \implies k_{BD} = -1$

$Y - Y_O = k_{BD} * (X - X_O) \iff Y - 11 = -X + 11 \iff X + Y - 22 = 0$

5) $AB$ $BD$ - $B$:

$ \begin{cases} 11X - 9Y - 2 = 0 \\ X + Y - 22 = 0 \end{cases} $

-11 :

$-20Y_B + 240 = 0 \implies Y_B = 12 \implies X_B + 12 - 22 = 0 \implies X_B = 10 \implies B = (10, 12)$.

6) $ABC$ :

$ S_{\triangle ABC} = \pm \frac{1}{2} \begin{vmatrix} X_C - X_A & Y_C - Y_A \\ X_B - X_A & Y_B - Y_A \end{vmatrix} \implies S_{\triangle ABC} = \pm \frac{1}{2} \begin{vmatrix} 21 - 1 & 21 - 1 \\ 10 - 1 & 12 - 1 \end{vmatrix} = \frac{1}{2} * (20*11 - 20 * 9) = 20 $

, $ABCD$ - $\implies \triangle ABC = \triangle ADC \implies S_{ABCD} = 2 * S_{\triangle ABC} = 2 * 20 = 40$.
