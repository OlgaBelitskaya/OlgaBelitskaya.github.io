$\bigg| \large{\frac{x^2 + x - 2 * a}{x + a}}$$ - 1 \bigg| \leq 2$

has no solutions on the interval $(1; 2)$.

Преобразуем неравенство:

Let's transform the inequality:

$\bigg| \large{\frac{x^2 + x - 2 * a}{x + a}}$$ - 1 \bigg| \leq 2$ $\iff \bigg| \large{\frac{x^2 - 3 * a}{x + a}}$$ \bigg| \leq 2$

$\begin {cases} \mid x^2 - 3 * a\mid \leq 2 * \mid x + a \mid \\ x \neq -a \end {cases} \iff \begin {cases} (x^2 - 3 * a)^2 \leq 4 * (x + a)^2 \\ x \neq -a \end {cases} \iff \begin {cases} (x^2 - 3 * a)^2 - 4 * (x + a)^2 \leq 0\\ x \neq -a \end {cases} \iff$

$\begin {cases} (x^2 - 3 * a - 2 * (x + a))(x^2 - 3 * a + 2 * (x + a)) \leq 0\\ x \neq -a \end {cases} \iff \begin {cases} (x^2 - 2 * x - 5 * a) * (x^2 + 2 * x - a) \leq 0 \\ x \neq -a \end {cases}$

Решений нет в двух случаях:

There are no solutions in two cases:

1) $x^2 - 2 * x - 5 * a > 0 \land x^2 + 2 * x - a > 0 \iff a < -0.2$

$\begin {cases} D_1 = 4 + 20 * a < 0 \implies a < -0.2 \\ D_2 = 4 + 4 * a < 0 \implies a < -1 \end {cases}$

For a = -0.2, the solutions of the inequality $(x^2 - 2 * x - 1) * (x^2 + 2 * x + 0.2) \leq 0$ do not intersect the segment (1; 2).

2) $x^2 - 2 * x - 5 * a < 0 \land x^2 + 2 * x - a < 0, \forall x \in (1;2)$

$a > 0.2 * x^2 - 0.4 * x \land a > x^2 + 2 * x, \forall x \in (1;2) \ (*)$
