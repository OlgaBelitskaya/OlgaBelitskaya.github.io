Задание 15¶

Решите неравенство:

$\log_{x + 2}^2 {(x - 18)^2} + 32 \leq 16 * \log_{x + 2} {(36 + 16 * x - x^2)}$¶

1) Область допустимых значений:

$\begin{cases} x + 2 > 0 \\ x + 2 \neq 1 \\ (x-18)^2 >0 \\ 36 + 16 * x - x^2 > 0\end{cases} \iff \ \begin{cases} x > -2 \\ x \neq -1 \\ x \neq 18 \\ x^2 - 16 * x - 36 < 0\end{cases} \iff \begin{cases} x > -2 \\ x \neq -1 \\ x \neq 18 \\ (x - 18) * (x + 2) < 0\end{cases} \iff \begin{cases} -2 < x < -1 \\ -1 < x < 18 \end{cases}$

2) Преобразования неравенства:

$\log_{x+2}^2 {(x - 18)^2} + 32 - 16 * \log_{x+2} {(36 + 16 * x - x^2)} \leq 0 \iff \log_{x+2}^2 {(18 - x)^2} + 32 - 16 * \log_{x+2} {((18 - x) * (x + 2))} \leq 0 \iff$

$\log_{x+2}^2 {(18 - x)^2} + 32 - 16 * \log_{x+2} {(x+2)} - 16 * \log_{x+2} {(18 - x)} \leq 0 \iff \log_{x+2}^2 {(18 - x)^2} + 32 - 16 - 16 * \log_{x+2} {(18 - x)} \leq 0 \iff$

$\log_{x+2}^2 {(18 - x)^2} - 16 * \log_{x+2} {(18 - x)} + 16 \leq 0 \iff \log_{x+2}^2 {(18 - x)^2} - 8 * \log_{x+2} {(18 - x)^2} + 16 \leq 0 \iff$

$ (\log_{x+2} {(18 - x)^2} - 4)^2 \leq 0 \iff log_{x+2} {(18 - x)^2} - 4 = 0 \iff log_{x+2} {(18 - x)} = 2$

$ (x + 2)^2 + x - 18 = 0 \iff x^2 + 5 * x - 14 = 0 \iff x = 2 \ \lor \ x = -7$ (не попадает в область допустимых значений)

15-го января планируется взять кредит в банке на 19 месяцев. Условия его возврата таковы:

  • 1-го числа каждого месяца долг возрастает на 3% по сравнению с концом предыдущего месяца;

  • со 2-го по 14-е число каждого месяца необходимо выплатить часть долга;

  • 15-го числа каждого месяца долг должен быть на одну и ту же величину меньше долга на 15-е число предыдущего месяца.

Сколько процентов от суммы кредита составляет общая сумма денег, которую нужно выплатить банку за весь срок кредитования?

Кредит возвращается $19$ равными платежами по $1/19$ от суммы кредита $Х$ и начисленных за весь период процентов:

  • в первом месяце процентов начисляется $0.03 * Х$
  • после уплаты части долга во втором месяце начисляется на $1/19$ меньше процентов $0.03 * \ \frac{18}{19} * Х$
  • в третьем месяце $0.03 * \ \frac{17}{19} * Х$

чтобы разность любых двух из них не была простой?

Чтобы разность между любыми двумя числами натурального ряда $1, 2, ..., 2009$ всегда не была простым числом,

она никогда не должна быть меньше 4.

    • (2008 - 4) / 4 + 1 = 502 числа.
