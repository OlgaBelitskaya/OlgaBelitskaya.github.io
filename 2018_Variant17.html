$AB = AC = AD \implies \triangle ABC: \angle ABC = \angle BCA = 56^o, \triangle ACD: \angle ACD = \angle CDA = 78^o$

$\angle BCD = \angle BCA + \angle CDA \implies \angle BAD = 360^o - \angle ABC - \angle BCD - \angle CDA = 360^o - 2 * (56^o + 78^o) = 92^o$

Ответ / Answer: 92¶
