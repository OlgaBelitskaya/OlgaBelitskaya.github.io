Shannon–Fano coding¶

В решении использован Алгоритм Шеннона — Фано.

С 1 начинается только одно кодовое слово последовательности `C-10`.

После сокращения на один символ эта буква будет кодироваться `C-1`, а вся последовательность по-прежнему декодируется однозначно.

The solution uses the Shannon-Fano Algorithm.

In the sequence, only one codeword `C-10` begins with 1.

After a reduction of one symbol, this letter will be encoded `C-1`, and the whole sequence is still possible to decode uniquely.

В терминологии сетей TCP/IP маской сети называется двоичное число, определяющее, какая часть IP-адреса узла сети относится к адресу сети, а какая — к адресу самого узла в этой сети.

Обычно маска записывается по тем же правилам, что и IP-адрес, — в виде четырёх байтов, причём каждый байт записывается в виде десятичного числа. При этом в маске сначала (в старших разрядах) стоят единицы, а затем с некоторого разряда — нули.

Адрес сети получается в результате применения поразрядной конъюнкции к заданному IP-адресу узла и маске.

import pandas

def find_years(number, array):
    df = pandas.DataFrame(index=range(1980,2018,1), 
                          columns=['max_inc_tonnage_per_vagon', 
                                   'min_out_tonnage_per_vagon'])
    df = df.fillna(0).astype('float32')
    
    for i in range(number):
        
        year = int(array[i].split()[0].split('.')[1])
        tonnage_per_vagon = int(array[i].split()[1])/int(array[i].split()[2])
        
        if tonnage_per_vagon < 0:
            if abs(tonnage_per_vagon) < df['min_out_tonnage_per_vagon'][year]\
            or df['min_out_tonnage_per_vagon'][year] == 0:
                df['min_out_tonnage_per_vagon'][year] = abs(tonnage_per_vagon)
 
        else:
            if abs(tonnage_per_vagon) > df['max_inc_tonnage_per_vagon'][year]:
                df['max_inc_tonnage_per_vagon'][year] = abs(tonnage_per_vagon)
            
    years = []
    for year in range(1980,2018,1):
        if df['max_inc_tonnage_per_vagon'][year] > \
        df['min_out_tonnage_per_vagon'][year]:
            years.append(year)
    return years
