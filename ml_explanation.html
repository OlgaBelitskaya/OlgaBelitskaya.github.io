One Neuron

Выберем функции для конструирования нейрона.

Находить значения этих функций будем сразу для набора чисел.

Для опыта классификации нейроном изображений будем использовать простой ряд из трех пикселей.
И всего три вида изображений:
темная полоса (номер 0), со светлым пикселем с краю (номер 1) и со светлым пикселем в середине (номер 2).

Возьмем набор из 10 изображений трех указанных видов.
Обозначим большинство из них правильно и одно - с ошибкой:
9-ое изображение состоит только из темных пикселей, нужно бы поставить номер 0, а стоит номер 1.

Определим стартовую позицию для подбора коэффициентов к переменным (оттенкам 3 пикселей) - набор 3 случайно выбранных чисел.

Выполним первое вычисление значений функции sigmoid(x).
Пока нейрон явно не готов классифицировать изображения правильно.
Столбец слева отличается от столбца справа.

Определим, насколько нейрон ошибся, вычитая из получившихся значений,
и "подскажем" направление для корректировки с помощью функции sigmoid_derivation(x).

После первой корректировки (столбец 1) стартовый набор коэффициентов изменился (столбец 2).

Повторим вычисления 100 раз. Это и есть "обучение" нейрона.

"Переведем" классификацию с помощью чисел от 0.5 до 1 в более простую (0, 1 или 2) и
сравним результат предсказаний с имеющимися реальными данными.
Электронный нейрон "исправил" нарочно допущенную ошибку в классификации 9-ого изображения.
