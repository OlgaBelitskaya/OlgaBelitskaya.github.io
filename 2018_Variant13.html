Задание 15¶

Решите неравенство:

$\log_{x + 2}^2 {(x - 18)^2} + 32 \leq 16 * \log_{x + 2} {(36 + 16 * x - x^2)}$¶

1) Область допустимых значений:

$\begin{cases} x + 2 > 0 \\ x + 2 \neq 1 \\ (x-18)^2 >0 \\ 36 + 16 * x - x^2 > 0\end{cases} \iff \ \begin{cases} x > -2 \\ x \neq -1 \\ x \neq 18 \\ x^2 - 16 * x - 36 < 0\end{cases} \iff \begin{cases} x > -2 \\ x \neq -1 \\ x \neq 18 \\ (x - 18) * (x + 2) < 0\end{cases} \iff \begin{cases} -2 < x < -1 \\ -1 < x < 18 \end{cases}$

2) Преобразования неравенства:

$\log_{x+2}^2 {(x - 18)^2} + 32 - 16 * \log_{x+2} {(36 + 16 * x - x^2)} \leq 0 \iff \log_{x+2}^2 {(18 - x)^2} + 32 - 16 * \log_{x+2} {((18 - x) * (x + 2))} \leq 0 \iff$

$\log_{x+2}^2 {(18 - x)^2} + 32 - 16 * \log_{x+2} {(x+2)} - 16 * \log_{x+2} {(18 - x)} \leq 0 \iff \log_{x+2}^2 {(18 - x)^2} + 32 - 16 - 16 * \log_{x+2} {(18 - x)} \leq 0 \iff$

$\log_{x+2}^2 {(18 - x)^2} - 16 * \log_{x+2} {(18 - x)} + 16 \leq 0 \iff \log_{x+2}^2 {(18 - x)^2} - 8 * \log_{x+2} {(18 - x)^2} + 16 \leq 0 \iff$

$ (\log_{x+2} {(18 - x)^2} - 4)^2 \leq 0 \iff log_{x+2} {(18 - x)^2} - 4 = 0 \iff log_{x+2} {(18 - x)} = 2$

$ (x + 2)^2 + x - 18 = 0 \iff x^2 + 5 * x - 14 = 0 \iff x = 2 \ \lor \ x = -7$ (не попадает в область допустимых значений)
