What is the minimum amount of memory (in Kbytes) you need to reserve, so that you can save any raster image with the size of 64x256 pixels, provided that the image can use 4 different colors? In the answer, write down only an integer, you do not need to write a unit of measurement.

Решение¶

  • $\log_{2}{4} = 2$ () - 4- .
  • $64 * 256 * 2 = 2^{6} * 2^{8} * 2 = 2^{15}$ () - 64 * 256 .
  • $2^{15} \ / \ 2^{3} = 2^{12}$ () - 64 * 256 .
  • $2^{12} \ / \ 2^{10} = 4$ () - 64 * 256 .

Decision¶

  • $\log_{2}{4} = 2$ (bit) - necessary to encode one pixel of the 4-color image.
  • $64 * 256 * 2 = 2^{6} * 2^{8} * 2 = 2^{15}$ (bit) - needed to encode an image with the size of 64 * 256 pixels.
  • $2^{15} \ / \ 2^{3} = 2^{12}$ (byte) - needed to encode an image with the size of 64 * 256 pixels.
  • $2^{12} \ / \ 2^{10} = 4$ (Kbytes) - needed to encode an image with the size of 64 * 256 pixels.
