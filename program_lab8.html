B, n n .

$X_1, X_2, ..., X_n$ :

$\large{\displaystyle{X_i = \begin{cases} 1, \ B_{ii} > B_{ij}, \ \forall j: 0<j<n+1 \land j \neq i\\ 0, \ B_{ii} \leq B_{ij}, \ \exists j: 0<j<n+1 \land j \neq i\\ \end{cases}, \ \forall i: 0<i<n+1 }}$

$\large{\displaystyle{S = \sum_{\substack{0<i<n+1 \\ 0<j<n+1}} B_{ij} }}$
