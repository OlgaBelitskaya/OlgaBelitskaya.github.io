Какой объём на диске (в Мбайт) будет занимать неупакованное 16-цветное изображение, содержащее изображение размером $2048*1024$ пикселей? В ответе запишите только целое число, единицу измерения писать не нужно.

What volume on the disk (in Mbytes) will be occupied by the unpacked 16-color image with the size of 2048 * 1024 pixels? In the answer, write down only an integer, you do not need to write a unit of measurement.

Решение¶

  • $\large {\log_{2}{16} = 4}$ (бита) - необходимо для кодирования одного пикселя 16-цветного изображения.
  • $\large {2048 * 1024 * 4 = 2^{11} * 2^{10} * 2^2 = 2^{23}}$ (бита) - необходимо для кодирования изображения размером 2048 * 1024 пикселей.
  • $\large {2^{23} \ / \ 2^{3} = 2^{20}}$ (байта) - необходимо для кодирования изображения размером 2048 * 1024 пикселей.
  • $\large {2^{20} \ / \ 2^{20} = 1}$ (Mбайт) - необходимо для кодирования изображения размером 2048 * 1024 пикселей.

Decision¶

  • $\large {\log_{2} {16} = 4}$ (bit) - necessary to encode one pixel of the 16-color image.
  • $\large {2048 * 1024 * 4 = 2^{11} * 2^{10} * 2^2 = 2^{23}}$ (bit) - needed to encode an image with the size of 2048 * 1024 pixels.
  • $\large {2^{23} \ / \ 2^{3} = 2^{20}}$ (byte) - needed to encode an image with the size of 2048 * 1024 pixels.
  • $\large {2^{20} \ / \ 2^{20} = 1}$ (Mbytes) - needed to encode an image with the size of 2048 * 1024 pixels.
